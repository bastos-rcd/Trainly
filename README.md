# Trainly

**Trainly** is a fully offline-first workout program application built with **React.js**, **TailwindCSS**, **IndexedDB**, and **Progressive Web App (PWA)** technologies. It allows users to manage their training routines directly on their device, **no server, no cloud, no account required**.

## Tech Stack

- **React.js**
- **TailwindCSS**
- **IndexedDB**

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/bastos-rcd/Trainly.git
   cd Trainly
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```
