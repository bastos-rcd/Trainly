# Trainly

Trainly is a web application designed to help users create, store and manage workout programs.

## Technologies

- **Next.js**: web application
- **Tailwind CSS**: styling
- **IndexedDB**: local database for storing workout programs

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/bastos-rcd/Trainly.git
   cd Trainly
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
